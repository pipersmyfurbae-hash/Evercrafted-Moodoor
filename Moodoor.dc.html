How matching works

A memory becomes geometry

No mood boards, no infinite generation. Every design already exists, already scored. Your memory just finds the one it belongs to.

1

Write the feeling

A porch, a person, a particular month. Whatever the memory smells and sounds like. The Emotional Vector System reads it into seven dimensions — warmth, energy, nostalgia, valence, intimacy, restraint, and seasonal affinity.

2

We place it on the grid

Your memory becomes a point in seven-dimensional space. Every wreath in the library is already a point too — scored the same way, before a single stem was cut. We measure the distance, deterministically.

3

Order it, or build it

The closest match arrives finished by hand in a numbered run — or as a digital blueprint with every stem, position, and count, so you can build it yourself. Same design, same score, two ways home.

The blueprint

Every stem placed by geometry, not guesswork

Before a single stem is cut, the design lives as a scored composition on a polar clock grid — balance, rhythm, negative space, and silhouette all resolved in the abstract. The blueprint is what gets built. There is no improvisation.

  • Clock-position placement on a numbered polar grid
  • Stem counts, layering order, and attachment points specified
  • Shopping list links to real, findable materials with substitutions
  • The same document our own builders follow — no separate "consumer" version
